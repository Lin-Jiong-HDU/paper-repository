## 论文精读

### 背景与前置知识

**联合嵌入预测架构（JEPA）** 是 Yann LeCun 提出的一类自监督学习框架。与传统的生成模型（需要重建像素）不同，JEPA 在潜空间（latent space）中进行预测：模型学会将输入映射到抽象表示，然后在表示空间中预测被遮蔽区域对应的表示。这避免了逐像素重建的冗余计算，让模型专注于学习高级语义特征。

**Vision Transformer（ViT）** 是将 Transformer 架构应用于视觉任务的方法。它将图像或视频切分为固定大小的 patch（块），将每个 patch 视为一个"词"，然后通过自注意力机制捕捉 patch 之间的关系。V-JEPA 2 使用 ViT 作为核心编码器。

**指数移动平均（EMA）** 是一种参数更新策略。在 V-JEPA 2 中，目标编码器不通过梯度更新，而是通过缓慢追踪上下文编码器的参数来更新。这保证了训练的稳定性，类似于 BYOL、DINO 等自监督方法中的教师-学生框架。

**VQ-VAE（向量量化变分自编码器）** 是一种将连续表示映射到有限离散码本的技术。V-JEPA 2-AC 使用它来学习离散的潜在动作空间，使得模型可以用有限的"动作词"来描述视频中的变化。

### 核心思想详解

V-JEPA 2 的核心理念可以用一个类比来理解：想象一个人在看大量的烹饪视频。他不需要有人告诉他每个动作叫什么名字，仅通过观看就能理解"搅拌""切菜""翻炒"这些动作的含义，并且能在脑中预测"下一步会发生什么"。V-JEPA 2 就是这样——通过"观看"海量视频来学习理解世界。

具体来说，V-JEPA 2 的设计包含三个递进的层次：

1. **观看并理解**（V-JEPA 2 预训练）：在海量视频上，通过遮蔽-预测的自监督任务学习视频的内在结构和物理规律。模型看到一部分视频帧，需要预测被遮蔽部分的潜在表示。

2. **理解和回答**（V-JEPA 2 + LLM 对齐）：将学到的视觉理解能力与语言模型结合，使其能够用自然语言回答关于视频内容的问题。

3. **理解并行动**（V-JEPA 2-AC）：在预训练基础上，用少量机器人操作视频教模型理解"动作"的概念，使其能够规划一系列动作来达成目标。关键创新在于：模型从互联网视频学到的物理世界知识可以迁移到机器人任务上，而只需要极少量的机器人数据。

### 方法逐步拆解

**第一阶段：V-JEPA 2 自监督预训练**

1. **视频采样与编码**：从数据集中采样短视频片段，将视频切分为时空 patch（例如 2 帧 × 16×16 像素的 patch），分别输入上下文编码器和目标编码器。

2. **遮蔽策略**：使用 tube masking（管道遮蔽），即同一空间位置在所有时间步都被遮蔽。这种策略强迫模型学习时序变化，而非仅靠单帧信息进行推断。遮蔽比例通常较高（约 75-90%）。

3. **潜在预测**：上下文编码器只处理可见的 patch，然后一个预测器网络根据可见 patch 的表示来预测被遮蔽位置的潜在表示。预测目标不是原始像素，而是目标编码器（EMA 更新）输出的表示。

4. **损失函数**：使用平滑 L1 损失（Smooth L1 Loss）衡量预测的潜在表示与目标编码器输出之间的差异。仅对被遮蔽位置计算损失。

5. **模型规模**：提供从 600M 到 6B 参数的多种规模，最大的 V-JEPA 2 模型在理解任务上表现最优。

**第二阶段：LLM 对齐**

1. 将 V-JEPA 2 的视觉编码器作为视觉骨干，通过投影层连接到大型语言模型（如 Llama 系列）。

2. 使用视频指令微调数据进行训练，使模型学会将视觉特征与自然语言描述对齐。

3. 在多个视频问答基准上进行评估，包括 PerceptionTest、TempCompass 等。

**第三阶段：V-JEPA 2-AC 动作条件世界模型**

1. **潜在动作学习**：从预训练的 V-JEPA 2 开始，添加一个动作编码器，使用 VQ-VAE 将视频中的变化量化为离散的潜在动作。具体地，给定两帧图像，动作编码器提取帧间的"变化向量"，然后量化到最近的码本向量。

2. **动作条件预测**：在预测器中加入潜在动作作为条件输入，使模型学会"执行这个动作后，世界会变成什么样"。

3. **后训练数据**：使用 Droid 数据集中约 62 小时的无标签机器人操作视频。模型只看视频，不需要任何动作标签或奖励信号。

4. **机器人规划**：给定目标图像，使用模型预测控制（MPC）：在潜空间中搜索最优的潜在动作序列，使预测的最终状态与目标最匹配，然后将潜在动作映射为实际机器人动作执行。

### 关键公式/算法解读

**JEPA 核心损失函数**：

```
L = Σ_{m∈M} SmoothL1(s_x(p_m), s_y(m))
```

其中：
- `M` 是被遮蔽的 patch 集合
- `s_x` 是上下文编码器 + 预测器对位置 `m` 的预测
- `s_y` 是目标编码器对位置 `m` 的输出（作为监督信号）
- 仅在被遮蔽位置计算损失，可见位置不参与

**EMA 更新**：

```
θ_target ← τ · θ_target + (1 - τ) · θ_context
```

其中 `τ` 是动量系数（通常接近 1），确保目标编码器平滑更新。

**潜在动作量化（VQ-VAE）**：

```
z = Encoder(f_t, f_{t+k})      # 从两帧提取变化向量
a = argmin_i ||z - c_i||       # 量化到最近的码本向量 c_i
```

将连续的动作空间映射到有限集合，使规划问题变为离散搜索。

**规划目标（MPC）**：

```
minimize Σ ||s_pred(t+i) - s_goal||  对动作序列求最小化
```

在潜空间中搜索使预测轨迹最接近目标表示的动作序列。

### 实验设计分析

**为什么选这些数据集？**

- **Something-Something v2**：专门测试运动理解的基准。与 ImageNet 等静态图像分类不同，它要求模型理解动作本身（如"把东西放在上面"vs"把东西移开"），而非仅靠物体外观判断。这直接验证 V-JEPA 2 是否真正学到了时序动态。

- **Epic-Kitchens-100**：第一人称视角的动作预测基准，要求模型在动作完成前预测即将发生什么。这考验模型对视频未来发展的推理能力。

- **PerceptionTest / TempCompass**：视频问答基准，测试模型对时空关系、物理状态变化的理解。选择这些基准验证 V-JEPA 2 + LLM 的多模态推理能力。

- **Droid 机器人数据集**：包含多个实验室、不同机器人的操作视频。选择它是因为数据多样性好，且完全不需要标签，验证自监督方法的实际可行性。

**实验结果表明什么？**

- 在 Something-Something v2 上达到 77.3% top-1 准确率，说明自监督方法可以在不使用任何人工标签的情况下学到强大的运动表示。
- 在 Epic-Kitchens-100 上以 39.7 recall@5 超越之前的专门模型，证明 JEPA 框架学到的表示包含丰富的时序预测信息。
- 在视频问答任务上达到 84.0（PerceptionTest）和 76.9（TempCompass），在 8B 参数规模下取得 SOTA，说明视觉-语言对齐策略有效。
- 零样本机器人操作实验的成功，说明从互联网视频学到的物理世界知识可以迁移到现实世界控制任务。

## 问题与动机

当前 AI 面临的核心挑战是：如何让模型像人类一样，主要通过观察来学习理解物理世界，并学会在物理世界中行动。传统方法要么依赖大量标注数据（成本高昂），要么仅局限于特定任务（泛化性差）。V-JEPA 2 的动机是探索一条"观察为主、交互为辅"的学习路径：先用海量无标签视频学习世界模型，再用极少量机器人数据让模型学会行动。

## 核心方法

V-JEPA 2 的方法可以分为三个阶段：

1. **自监督视频预训练**：基于 JEPA 框架，使用遮蔽预测任务在超过 100 万小时互联网视频上训练。核心创新是使用 tube masking 策略和高遮蔽比例，强迫模型学习深层时序动态。

2. **视觉-语言对齐**：将预训练的 V-JEPA 2 与 LLM 对齐，实现视频问答能力。

3. **动作条件世界模型**：通过 VQ-VAE 学习潜在动作空间，在约 62 小时无标签机器人视频上后训练，实现零样本机器人规划。

## 实验结果

- **数据集**: Something-Something v2, Epic-Kitchens-100, PerceptionTest, TempCompass, Droid, Ego4D, ActivityNet-QA, 等
- **主要指标**: Top-1 准确率（分类）, Recall@5（动作预测）, 准确率（视频问答）, 成功率（机器人操作）
- **关键结果**: SSv2 77.3% top-1; EK-100 39.7 recall@5; PerceptionTest 84.0; TempCompass 76.9; 零样本机器人操作成功
- **对比方法**: VideoMAE, V-JEPA (v1), InternVideo2, Gemini 1.5 Pro, GPT-4o 等

## 局限性

- 视频处理长度有限，当前模型只能处理短片段（约几秒到十几秒），无法理解长视频的完整叙事或复杂时序关系。
- 机器人实验仅限于相对简单的 pick-and-place 任务，尚未验证在更复杂、长周期操作中的效果。
- 世界模型的物理模拟精度有限，对于精细操作（如灵巧手操控）可能不足。
- 潜在动作空间的表达能力受码本大小限制，可能无法覆盖所有可能的动作。
- 模型规模虽然达到 6B 参数，但与当前最大的语言模型相比仍有差距，进一步扩展可能面临计算瓶颈。
- 零样本机器人实验的环境条件仍然相对可控（桌面操作），在更开放环境中的鲁棒性有待验证。

## 个人思考

V-JEPA 2 是 JEPA 系列研究中一个重要的里程碑，它展示了 LeCun 一直倡导的"世界模型"路线的实际可行性。论文最有价值的贡献不在于单个基准上的数字，而在于验证了一个关键假设：从大规模无标签视频中学到的物理世界知识可以迁移到机器人控制任务。这意味着我们可能不需要为每个机器人任务都收集专门的训练数据，而是可以利用互联网上已有的海量视频资源。

V-JEPA 2-AC 仅用 62 小时无标签机器人视频就实现了零样本操作，这一点令人印象深刻。不过，从更长远的角度看，当前的成功仍局限于简单场景。要让世界模型真正驱动机器人在复杂现实中自主行动，可能还需要在以下方向突破：更长的时序建模、更精细的物理模拟、以及更强大的规划算法。
