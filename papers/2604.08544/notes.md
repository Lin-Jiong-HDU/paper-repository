# SIM1: Physics-Aligned Simulator as Zero-Shot Data Scaler in Deformable Worlds

## 基本信息

- **标题**: SIM1: Physics-Aligned Simulator as Zero-Shot Data Scaler in Deformable Worlds
- **作者**: Yunsong Zhou, Hangxu Liu, Xuekun Jiang, Xing Shen, Yuanzhen Zhou, Hui Wang, Baole Fang, Yang Tian, Mulin Yu, Qiaojun Yu, Li Ma, Hengjie Li, Hanqing Wang, Jia Zeng, Jiangmiao Pang
- **发表日期**: 2026年4月10日
- **arXiv ID**: 2604.08544
- **arXiv 链接**: <https://arxiv.org/abs/2604.08544>
- **领域/关键词**: robotics, deformable objects, 物理对齐, 柔性物体, 零样本, sim-to-real, 数据引擎

## 一句话总结

> SIM1 是首个专门为柔性物体（如衣物）操作设计的物理对齐仿真数据引擎，通过精确的场景数字化、稳定化的柔性体解算器和扩散轨迹生成，实现纯合成数据训练的策略零样本迁移到真实机器人，达到 15:1 的数据性价比（15个合成样本 ≈ 1个真实样本）。

## 论文精读

### 背景与前置知识

**柔性物体操作（Deformable Object Manipulation）** 是机器人学中极具挑战性的问题。与刚性物体不同，柔性物体（如T恤、毛巾、布料）在操作过程中形状、接触点和拓扑结构会持续变化。这导致传统刚性物体操作的方法（如抓取点检测、简单的拾取放置）完全不适用。

**Real-to-Sim-to-Real (R2S2R)** 是一种从真实到仿真再到真实的流水线范式：先对真实环境进行数字化重建（Real→Sim），在仿真中生成和扩增数据，然后将训练好的策略部署回真实世界（Sim→Real）。

**顶点块下降法（Vertex Block Descent, VBD）** 是一种用于柔性体仿真的数值求解方法。它将布料网格的顶点视为粒子，通过迭代优化更新顶点位置以最小化能量函数。

**应变约束（Strain Constraint）** 在布料仿真中用于控制顶点之间的最大允许拉伸距离，防止布料出现不真实的大形变。

**扩散模型用于轨迹生成**：与图像生成类似，扩散模型可以学习从随机噪声逐步去噪到合理的机器人操作轨迹。

### 核心思想详解

SIM1 的核心理念是：仿真失败不是因为它是"合成的"，而是因为它"没有接地气"（ungrounded）。在柔性物体操作中，几何、物理和运动三个层面的微小不匹配都会被柔性体的形变放大，导致仿真与真实之间不可逾越的鸿沟。

**类比**：想象你在教一个人叠衬衫。如果用一个完全不同的衬衫（不同的尺寸、材质、版型）来教学，他学到的技巧可能完全用不上。同样，如果仿真中的"衬衫"在形状、材质弹性、重量分布上都与真实衬衫不一致，那么在仿真中学到的叠衣技巧在真实世界中必然失败。SIM1 的做法是：先精确测量真实衬衫的每一个参数（三维扫描、材质测试），然后在仿真中创建一个完全一致的"数字克隆"，再基于这个精确的克隆来设计和扩增训练数据。

SIM1 在三个层面实现物理对齐：
1. **几何对齐**：亚毫米精度的3D扫描重建
2. **动态对齐**：通过稳定化的柔性体解算器和参数校准，让仿真中的布料行为与真实一致
3. **运动对齐**：通过结构化轨迹分解和扩散生成，产生类似人类的自然操作动作

### 方法逐步拆解

**第一步：SIM1-Scene 真实场景数字化**

**柔性物体数字化**：使用专业3D扫描仪（EinScan Rigil Pro）对衣物进行高精度扫描。衣物穿在人体模型上保持自然形态，多视角RGB图像和LiDAR扫描融合生成密集点云。通过人工分割去除人体模型点云，只保留衣物。然后经过泊松重建、孔洞填充、平滑和网格重划分，获得亚毫米精度的带纹理3D网格。

**机器人数字化**：从CAD模型生成URDF文件，精确描述机器人的运动学结构、关节限制和碰撞几何体。在仿真中导入URDF，确保每个臂的自由度和工作空间与真实硬件完全一致。

**环境数字化**：从公开3D模型库或手工创建，按真实世界测量值缩放尺寸，保持物理一致性。

**第二步：SIM1-Sim 变形稳定的物理仿真**

这是论文的核心技术贡献。现有柔性体解算器在离线仿真中可以产生准确的形变，但在与机器人实时交互时，力传播不足和应变更新延迟导致过度拉伸、不稳定接触和仿真与真实的不一致。

**稳定化解算器**：基于增强型顶点块下降法（AVBD），在标准VBD迭代之后增加应变约束检查。对于每个网格边，如果两个顶点之间的距离超过允许的拉伸阈值（$(1+\xi)l_0$），则激活一个虚拟弹性约束，注入额外的张力到VBD优化中，加速收敛到物理合理的构型。

应变约束：$C(e) = \|e_i - e_j\| - (1+\xi)l_0 \leq 0$

当 $C(e) > 0$ 时激活惩罚能量：
$$E_{strain}(e) = \begin{cases} \frac{1}{2}k^{(n)}C(e)^2 + \lambda^{(n)}C(e), & C(e) > 0 \\ 0, & \text{otherwise} \end{cases}$$

**参数校准**：通过双向同步的仿真基础设施，专家操作真实双臂机器人执行操作序列，机器人的关节状态被流式传输到仿真器，让仿真孪生复现相同的运动。通过视觉对比评估差异，迭代调整物理参数（密度、杨氏模量、泊松比、摩擦系数等），直到仿真的行为与真实在操作层面一致。

**第三步：SIM1-DataGen 结构化柔性体操作合成**

**轨迹分解**：将示教轨迹分割为"交互段"（抓取构型、接触状态）和"运动段"（中间过渡）。交互段保持不变并池化，通过随机选取不同组合来增加多样性。

**扩散运动生成**：对于交互段之间的过渡，使用条件扩散模型生成平滑、符合运动学的中间轨迹。模型以边界位姿 $(p_s, p_e)$ 和机器人历史状态 $h$ 为条件，预测中间运动。

**有效性检查**：两步过滤——先用基于粒子状态的轻量规则过滤明显无效的轨迹，再用基于视频的二分类器判断剩余轨迹的质量。

**视觉随机化**：在Blender中对有效轨迹进行材质、光照和相机参数的外观随机化，每个轨迹生成多个视觉变体。

### 关键公式/算法解读

**应变约束公式（公式1）**：
$$C(e) = \|e_i - e_j\| - (1+\xi)l_0 \leq 0$$

这是仿真中最关键的公式。$e_i, e_j$ 是网格边的两个顶点，$l_0$ 是边的初始长度（静止长度），$\xi$ 是最大允许拉伸比。当边的实际长度超过 $(1+\xi)l_0$ 时，约束被激活，施加力将其拉回。

**应变能量公式（公式2）**：
$$E_{strain}(e) = \frac{1}{2}k^{(n)}C(e)^2 + \lambda^{(n)}C(e)$$

这是一个带拉格朗日乘子的增广拉格朗日函数。$k^{(n)}$ 是第 $n$ 次牛顿迭代时的惩罚刚度，$\lambda^{(n)}$ 是累积约束力的拉格朗日乘子。可以这样理解：$\frac{1}{2}kC^2$ 像一个弹性系数为 $k$ 的弹簧，拉伸越远拉力越大；$\lambda C$ 是一个额外的"推动力"，帮助更快地将顶点推回合理位置。

**参数更新公式（公式3）**：
$$k^{(n+1)} = \min(k_{max}, k^{(n)} + \beta|C(e)|), \quad \lambda^{(n+1)} = \lambda^{(n)} + k^{(n)}C(e)$$

惩罚刚度 $k$ 逐步递增（最多到 $k_{max}$），拉格朗日乘子 $\lambda$ 累积之前的约束力。这种渐进式递增避免了突然施加过大力导致的数值不稳定性。

### 实验设计分析

**任务设计**：主要评估任务为T恤折叠，包含抓取、提起、拉直、翻转、折叠等一系列长时间（>20秒）操作步骤。

**评估设置**：
- **域内**：训练时使用的空间布局、衣物、桌面纹理、光照和相机配置
- **泛化**：空间平移（≤8cm/±15°旋转）、纹理（新桌面和衣物材质）、光照（方向和强度变化）、视角（±5°俯仰角）

**主要发现**：
1. **仿真 vs 真实数据性能相当**：在相同数据量下（200条），纯合成数据训练的策略（𝜋0.5 成功87%）与真实数据训练的策略（97%）差距仅10%。这意味着物理对齐的仿真可以提供接近真实世界的监督信号。
2. **泛化优势**：在空间平移、纹理变化和光照扰动下，纯合成数据训练的策略分别超越真实数据基线50%、13%和47%。仿真可以覆盖更广泛的变化范围。
3. **预训练混淆分析**：从零训练（不使用预训练权重）的 𝜋0.5 在真实数据上完全失败（0%），而合成数据训练达到76%。这证明了性能提升来自生成数据的质量而非预训练先验。
4. **数据规模分析**：域内评估中，1个真实样本 ≈ 15个合成样本；纹理泛化中，1个真实样本 ≈ 5个合成样本。随着合成数据量增加，性能持续提升，而真实数据增长受限。

**消融实验**：
- 基线（MimicGen风格轨迹切割）→ 通过率0%，完全无法生成有效数据
- +轨迹分解 → 通过率65%，但任务成功0%（缺乏物理一致性）
- +扩散生成 → 域内47%，泛化仍弱
- +变形稳定解算器 → 域内67%，平均76%，大幅提升

## 问题与动机

柔性物体操作是机器人学习中数据需求最旺盛的领域之一，因为其形状、接触和拓扑共同演化，远超刚性物体的变异性。现有 sim-to-real 流水线基于刚性物体假设，在几何、动态和运动三个层面都不适合柔性物体。SIM1 的核心理念是：仿真的第一原则是"接地气"（grounding），只有在物理与现实对应之后，规模化才有价值。

## 核心方法

SIM1 采用 R2S2R 范式的三阶段对齐：1）几何对齐——通过高精度3D扫描和网格后处理，重建亚毫米精度的带纹理仿真资产；2）动态对齐——设计基于 AVBD 的变形稳定化解算器，引入显式应变约束防止过度拉伸，通过双向同步仿真基础设施进行视觉校准；3）运动对齐——通过交互-运动解耦的结构化轨迹分解、扩散运动生成和两级有效性过滤，合成高质量柔性物体操作轨迹。

## 实验结果

- **数据集**: 1000条真实轨迹；200条仿真遥操作种子轨迹 → 扩增至2000-10000条合成轨迹
- **主要指标**: 任务成功率（30次试验/配置），域内+4种泛化设置
- **关键结果**: 零样本 T恤折叠 ✓ 90%（𝜋0.5）和76%（𝜋0）；泛化增益 +50%/+56%（空间/光照）；1:15 数据等价比
- **对比方法**: 真实数据训练基线 vs 仿真遥操作数据 vs 仿真生成数据；𝜋0.5 vs 𝜋0；从零训练 vs 微调

## 局限性

1. **材料标定依赖专家**：每个柔性物体的物理参数标定需要专家引导的迭代调整，无法完全自动化。
2. **扫描成本**：需要专业3D扫描设备，增加了实际部署的门槛。
3. **任务范围**：当前主要验证了T恤折叠任务，虽然支持更广泛的任务类型，但全面验证有待展开。
4. **成像假设**：几何重建假设衣物处于静态挂载状态，对于完全散落、折叠状态的衣物处理能力有限。

## 个人思考

SIM1 在柔性物体操作这个极难的问题上取得了令人印象深刻的成果，其最核心的贡献是证明了"物理对齐的仿真可以作为可靠的数据源"这一观点。在刚性物体领域，人们往往可以通过大量数据来"平均掉"仿真的不精确性；但在柔性物体领域，每个仿真细节的偏差都会被形变放大，因此从"根"上对齐仿真与真实是唯一可行的路径。

变形稳定化解算器的设计非常有启发性。它不是在物理引擎层面大刀阔斧地改造，而是在现有VBD求解器的基础上增加了一个精巧的"安全网"——应变约束。这种"轻干预"的思路使得它可以在保持计算效率的同时显著改善仿真质量。

15:1的数据等价比是一个惊人的数字——或者说，合成数据的经济效益是真实数据的15倍。考虑到真实数据采集需要专业操作员、专用硬件和时间成本，这个比例在实际应用中可能更大。这也暗示着，对于柔性物体操作这类数据密集领域，高质量的仿真数据引擎可能比大规模真实数据采集更具成本效益。

一个有趣的方向：是否可以将物理参数校准过程自动化？当前依赖人的视觉判断来调参，如果能构建一个自动化的"仿真-真实差异检测"系统，通过优化算法自动搜索最佳物理参数，SIM1的自动化程度将大大提高。
